# 🏠 Advanced Real Estate Analytics

ML-powered real estate price prediction with feature engineering, geospatial analysis, and market insights.
Predicts house prices using XGBoost with 94% R² on 13,000+ properties.

## 🎯 Advanced Features

- **Price Prediction:** XGBoost with 94% R² score
- **Feature Engineering:** 50+ engineered features from location, property, market data
- **Geospatial Analysis:** Neighborhood clustering, location value mapping
- **Market Insights:** Price trends by neighborhood, ROI analysis
- **Property Scoring:** Investment potential (0-100)
- **Comparable Analysis:** Find similar properties in area
- **Interactive Maps:** Plotly maps with price heatmaps

## 📊 Tech Stack

- **ML:** XGBoost, Scikit-learn, advanced feature engineering
- **Geospatial:** Latitude/longitude clustering
- **Data:** 13,000+ realistic property records
- **Visualization:** Plotly maps, Seaborn heatmaps
- **Dashboard:** Streamlit with Plotly maps
